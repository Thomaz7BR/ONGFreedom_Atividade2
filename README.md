ONG Freedom – Atividade 2

Este projeto é a segunda entrega da disciplina Linguagem de Estilo: Construindo Interfaces Web Modernas com CSS3.
O objetivo desta entrega foi aplicar CSS3 para transformar a estrutura HTML da Entrega I em uma interface visual profissional, responsiva e acessível, utilizando técnicas modernas de design e estilização.

O projeto inclui layouts específicos para diferentes tipos de conteúdo, como seções de voluntários, doações e cadastro.

Sistema de Design

Paleta de cores: 8 cores (primárias, secundárias e neutras) definidas em variáveis CSS.

Tipografia hierárquica: 5 tamanhos de fonte.

Sistema modular de espaçamento: 8px, 16px, 24px, 32px, 48px, 64px.

Leiautes Responsivos

Estrutura principal usando CSS Grid.

Componentes internos alinhados com Flexbox.

Menu mobile com hamburger funcional.

Breakpoints para diferentes tamanhos de tela.
